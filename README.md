# Diametric Purchase Task Board

Static frontend for the Purchase Task Board dashboard, hosted via GitHub Pages.

- `index.html` — the dashboard (calls the deployed Google Apps Script backend for data)
- Backend (`Code.gs`) lives separately in Google Apps Script — see the "Diametric Purchase Task Board" internal reference for that file.

## Updating the live site
1. Edit `index.html` locally (or ask Claude for changes).
2. Commit and push to the `main` branch.
3. GitHub Pages redeploys automatically within a minute or two.
